# AI Image Generator with Azure AI Foundry (FLUX.2-pro)

A simple Python application that generates images using the FLUX.2-pro model deployed in Azure AI Foundry.

## Features

- Generate images from text prompts
- Azure AI Foundry integration
- FLUX.2-pro image generation
- Saves generated images locally

## Technologies

- Python
- Azure AI Foundry
- FLUX.2-pro
- Requests
- dotenv

## Example Prompt

Robot eating spaghetti in a restaurant while a human waiter is serving.

## Usage

1. Configure `.env`
2. Run:

```bash
py image-client.py
```

3. Enter a prompt
4. Generated image is saved locally
